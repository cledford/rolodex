# rolodex

路           Create a single-page CRUD application using C#, WebAPI (.Net Core 2.2), front end of your choice, with data being stored in a relational database.

路           The application would be used by approximately 20,000 users daily, and we expect each of those users to maintain ~1,000 contact records.

路           A user should be able to interact with a scrollable list of their business contacts, as well as adding, editing, and deleting their contacts.

路           Each business contact record contains basic information such as email, name and address. The application should access a simple database through a RESTful service. The UI is undefined.
